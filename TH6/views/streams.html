Demo 4 Loại Stream trong NodeJS

1. Readable Stream

Đọc file bằng fs.createReadStream() và ép trình duyệt tải về file log.

2. Writable Stream

Server dùng fs.createWriteStream() để ghi dữ liệu bạn nhập vào file output.txt.

3. Transform Stream

Đọc file story.txt, đi qua ống biến đổi (In hoa) rồi mới trả về trình duyệt.

4. Duplex Stream (Echo)

Gửi dữ liệu lên, Server vừa đọc (read) vừa dội ngược ngay lại trình duyệt (write).
